# 멋쟁이사자처럼 2회차 과제

## 자바스크립트 언어가 다른 언어와 다른 특징 3가지를 작성해주세요

첫번째로 자바스크립트는 웹 브라우저가 이해할 수 있는 유일한 언어입니다. 두번째로 코드를 실행하는 시점에서 필요한 부분을 즉석에서 컴파일 하는 방식인 JIT 방식을 사용합니다. 세번째로 자바스크립트는 주로 Node.js 서버를 통해서 서버 측에서도 실행이 가능합니다.

## 변수에 대해 설명하고 let, const, var 차이점을 작성해주세요

var은 중복 선언과 재할당이 가능하며 같은 파일 내에서 어디에 선언하든 어디서든 참조가 가능합니다.
let은 var과 다르게 중복 선언이 불가능하지만 재할당이 가능하고 블록 단위 스코프인 {}안에서만 참조가 가능합니다.
const는 수식에서 변하지 않는 값인 상수를 선언하며 중복 선언과 재할당이 모두 불가능합니다. 또한 반드시 선언 시 값을 할당해야 하며 let과 동일하게 블록 단위 스코프인 {} 안에서만 참조가 가능합니다.

## ==과, ===의 차이점을 작성해주세요

=='은 두 피연산자의 값이 같은지를 판단하며, 타입이 다를 경우 타입 강제 변환을 수행한 후 비교합니다. '==='은 두 피연산자의 값뿐만 아니라 자료형까지 동일한지를 판단하며, 타입 강제 변환을 수행하지 않습니다.

## var 변수 타입을 지양해야하는 이유를 말해주세요

코드를 작성하다보면 여러 변수를 선언하게 되는데 중복 선언과 재할당이 가능한 var 함수를 사용할 시 코드 다른 곳에서 쉽게 변수의 값이 변할 수 있기 때문입니다.

## function foo() {}와 var foo = function() {} 사이에서 foo 사용의 차이에 대해 작성해주세요

function foo() {}는 함수 선언으로 어디서든 호출이 가능하고 var foo = function() {}은 함수 표현식으로 선언 이후에만 사용이 가능하다는 차이가 존재합니다.

## 화살표 => 함수 문법에 대한 사용 예시를 작성해주세요. 이 문법은 다른 함수와 어떻게 다른지 작성해주세요

let sum = (a,b) => a+b;
화살표 함수는 보다 간단하게 함수를 정의할 수 있는 문법입니다. 일반 함수와 달리 this를 자체적으로 가지지 않고 상위 스코프의 This를 사용하며, 생성자 함수로 사용할 수 없습니다.

## spread 문법에 대해 작성해주세요

spread 문법은 배열이나 문자열과 같이 반복이 가능한 문자를 0개 이상의 인수 또는 요소로 확장할 수 있게 해주는 문법입니다.
const a = [1,2,3]
const b = [...a,4,5]

# 주제 정리🦁

## 배열

배열은 여러개의 데이터를 하나의 변수에 순서대로 저장할 수 있는 자료구조이다. 각 요소는 인덱스를 통해 접근할 수 있으며, 다양한 데이터 타입을 함께 저장할 수 있다는 특징이 있습니다. 또한 push, pop과 같은 것 뿐만 아니라 map, filter와 같은 메서드를 통해 데이터를 효율적으로 처리할 수 있습니다. 이러한 배열의 특징은 반복적인 데이터 처리와 목록 관리에 매우 유용하며 코드의 간결성과 가독성을 높이는 데 기여합니다.

## 객체

객체는 속성과 값의 쌍으로 이루어진 프로퍼티들의 집합으로, 관련된 데이터를 하나로 묶어 표현할 수 있는 자료구조입니다. 객체의 값에는 문자열, 숫자, 배열, 함수 등 다양한 데이터 타입이 포함될 수 있어 매우 유연한 구조를 가집니다. 또한 점 표기법과 대괄호 표기법을 통해 프로퍼티에 접근할 수 있으며 동적으로 속성을 추가하거나 수정하는 것도 가능합니다.

## 콜백 함수

콜백 함수란 다른 함수의 인자로 전달되어 특정 시점에 실행되는 함수를 의미합니다. 자바스크립트에서는 함수를 값처럼 다룰 수 있는 특성을 가지기에 콜백함수를 자유롭게 활용할 수 있습니다. 콜백 함수를 사용하면 프로그램의 실행 흐름을 유연하게 제어할 수 있고, 코드의 재사용성과 확장성을 높일 수 있는 장점이 존재합니다.
