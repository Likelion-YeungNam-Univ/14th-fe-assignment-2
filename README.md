## 변수에 대해 설명하고 let, const, var 차이점을 작성해주세요
변수란 데이터를 저장하기 위해 사용하는 이름이 붙은 공간이며 js에서는 위 3가지 변수 선언 방식이 있다. let은 제일 기본적인 변수 선언이며 중복 선언이 불가능하고 재할당이 가능하다. const는 상수를 선언하는 데에 쓰이며 중복 선언과 재할당 둘다 불가능하기에 반드시 선언 시 값을 할당해주어야 한다. var은 let 키워드가 나온 이후 권장되지 않지만 초기 기본 변수 선언 방식이고 중복 선언과 재할당이 가능하다.

전자는 함수 선언문이며 어디서든 호출이 가능하고, 후자는 함수 표현식으로 선언 이후에만 사용이 가능하다. 호이스팅(수나 함수의 선언이 코드의 최상단으로 끌어올려지는 것처럼 동작하는 현상) 차이가 있다. 


## 화살표 => 함수 문법에 대한 사용 예시를 작성해주세요. 이 문법은 다른 함수와 어떻게 다른지 작성해주세요
function Checkout(name){
    alert("안녕히 가세요~"+ name+ "님!" );
}
Checkout("김민서");
-------------------------------------------
const Checkout = (name) => { 
alert("안녕히 가세요." + name + "님!"); 
}
Checkout("홍길동");
-------------------------------------------
화살표 함수 사용시 function 키워드를 사용해서 함수를 만드는 것보다 더 간단히 함수를 표한할 수 있다. 


## spread 문법에 대해 작성해주세요
spread 문법이란 ...기호를 사용해서 배열이나 객체의 요소를 펼쳐서 사용하는 문법이다. 
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5];
console.log(arr2); // [1, 2, 3, 4, 5]

const user = { name: "민서", age: 20 };
const newUser = { ...user, age: 21 };
console.log(newUser); // { name: "민서", age: 21 }


# 주제 정리🦁

## 주제1 (ex. 배열)
여러 개의 데이터를 하나의 그룹처럼 묶어서 저장하고 인덱스로 접근할 수 있는 자료구조이다. 배열관련 함수에는 배열의 값을 하나씩 바꿔서 새 배열을 만드는 map()함수, 조건에 맞는 요소만 추출하고 싶을 때 사용하는 filter()함수, 배열에 하나라도 조건을 만족하는 게 있는지 확인할 때 사용하는 some()함수가 있다. 

## 주제2 (ex. 객체)
객체는 key:value 한 쌍으로 묶은 덩어리들로 이루어지는데 이 한 쌍을 프로퍼티라고 한다(ex. name: "피카츄", type: "전기" 하나하나가 프로퍼티). 점 표기법, 대괄호 표기법이 있고 관련 함수로는 대표적으로 Object.keys(), Object.values(), Object.entries()가 있다. 배열을 객체로 변환하는 Object.assign()도 있다.

## 주제3 (ex.화살표 함수)
function 대신 =>를 사용하는 간결한 함수 표현 방식, 생성자 함수로 사용이 불가능하다. ++ 화살표 함수는 자체적인 this를 가지지(바인딩) 않고, 외부 스코프의 this를 그대로 사용하는 특징이 있다. 생성자 함수로 사용 불가능하다.
