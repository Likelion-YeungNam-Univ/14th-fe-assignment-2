## 변수에 대해 설명하고 let, const, var 차이점을 작성해주세요

var 함수는 중복 선언과 재할당이 가능하며 같은 파일 내에서 어디에 선언하든 어디서든 참조가 가능합니다.
let 함수는 var 함수와 다르게 중복 선언이 불가능하지만 재할당이 가능하고 블록 단위 스코프인 {}안에서만 참조가 가능합니다.
const는 수식에서 변하지 않는 값인 상수를 선언하는 함수로 중복 선언과 재할당이 모두 불가능합니다. 또한 반드시 선언 시 값을 할당해야 하며 let 함수와 동일하게 블록 단위 스코프인 {} 안에서만 참조가 가능합니다.
