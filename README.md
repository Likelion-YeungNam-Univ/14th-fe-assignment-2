## 자바스크립트 언어가 다른 언어와 다른 특징 3가지를 작성해주세요
1. 다른언어와 다르게 자바스크립트는 JIT 컴파일 실행방식을 사용하는데 JIT 컴파일 방식이란 코드를 실행하는 시점에서 필요한 부분을 즉석에서 컴파일 하는 방식을 사용하고있습니다.
2. 다른언어들은 타입설정을 직접 설정해야하지만, 자바스크립트에서는 동적 타입 언어로 코드가 실행될때 타입이 결정됩니다.
3. 다른어어들은 별도의 설치과정을 거치고 실행해야하지만, 자바스크립트는 별도의 설치과정없이 모든 웹 브라우저에서 짠 코드가 웹브라우저 내에서 곧바로 실행됩니다.

## 변수에 대해 설명하고 let, const, var 차이점을 작성해주세요
변수는 숫자형(Number)와 문자형(String), 논리형(Boolean)이 있다. 숫자형은 말 그대로 숫자(1,2,4.5)와 같은 숫자를 뜻하고, 문자형은 따옴표("")로 감싸진 문자형태를 말합니다. 논리형은 True,False로 이루어진 0,1 중 하나의 값을 가집니다.

var는 중복선언과 재할당이 가능하고, let은 중복선언이 불가능하지만 재할당은 가능합니다. 하지만 const 선언은 중복선언과 재할당이 모두 불가능하다. 따라서 const는 선언시 반드시 값을 할당해주어야합니다.
var는 어디에 선언하든 어디서든 참조가 가능하지만, let과 const는 블록 단위 스코프인 {}안에서 선언 시 블록 안에서만 참조 가능합니다.

let,var,const 모든 선언은 호이스팅이됩니다. 호이스팅은 해당 스코프의 선두로 옮긴 것처럼 동작하는 특성을 말합니다.
var선언은 호이스팅이 되어서 이전에 var선언한 변수를 출력하면 undefined가 출력되고, let이나 const 변수는 에러가 발생합니다.

## ==과, ===의 차이점을 작성해주세요
== 연산자는 두개의 피연산자의 값이 같은지 비교합니다.
=== 연산자는 두개의 피연산자의 값을 비교 + 자료형까지 비교합니다.

## var 변수 타입을 지양해야하는 이유를 말해주세요
var 변수는 중복선언과 재할당이 가능하고 또 블록단위 스코프를 무시하기때문에 원래의 있던 값이 블록단위 스코프내에서 같은 이름으로 선언하면 블로단위 스코프 밖에 있는 값이 같이 바뀌는 경우가 발생할 수 있습니다. 

## function foo() {}와 var foo = function() {} 사이에서 foo 사용의 차이에 대해 작성해주세요
function foo() {} 는 함수명 foo라는 함수를 선언할때 사용하는 방법이고 var foo = function() {} 은 function() {} 은 var foo라는 변수에 익명함수를 할당하는 방식입니다.
그래서 function foo() {}도 호이스팅이 되는데, 선언문 이전에 foo()를 출력해도 정상출력이 되지만, var foo = function() {} 은 var 변수의 호이스팅 규칙을 따르기 때문에 이 선언문 이전에 출력하면 undefined가 출력됩니다. 선언문 이후에 호출하면 정상적으로 출력됩니다.

## 화살표 => 함수 문법에 대한 사용 예시를 작성해주세요. 이 문법은 다른 함수와 어떻게 다른지 작성해주세요
만약 함수가 간단한경우 예를 들어서 함수 몸체 내의 코드가 한줄이고 단순한 return문만 있는경우, 화살표 => 함수로 return을 생략할 수 있습니다.
ex) function sum=(a,b){
  return a+b;
  }
를 const sum = (a,b) => a+b;로 줄일 수 있습니다.

이 문법은 다른 함수와 다르게 매개변수가 하나이면 매개변수의 괄호를 생략이 가능하고 함수 몸체내의 코드가 한줄이고 return문만 있으면 return 키워드를 생략할 수 있습니다.

## spread 문법에 대해 작성해주세요
spread 문법은 배열이나 객체,문자열 같이 반복 가능한 데이터의 껍데기를 벗기고 안의 내용물을 펼쳐서 전개해주는 역할을 해줍니다.
예를들어서 배열이나 객체를 복사할때 [...객체명] 이나 [...배열명]으로 안의 내용물을 쉽게 복사할 수 있습니다.

# 주제 정리🦁

## 콜백 함수
콜백함수가 어떻게 동작하는지 수업시간에 제대로 이해하지못하였는데, 인터넷을 찾아보면서 공부하였습니다.
콜백함수는 함수에서 매개변수로 callback 함수를 인자로 받는데 이 callback 함수 인자는 함수의 모든 실행이 다 끝나면 callback()으로 
콜백함수를 호출합니다. 전달할때는 callback함수에서 함수명만 전달하면 되어서 매우 편리하고 중요한 작업인것같습니다.

## 화살표 함수
화살표 함수가 이때까지 배운적이 없어서 조금 어려워서 강의자료를 다시 보면서 공부하였습니다.
화살표함수로 함수 선언을 하면 function으로 선언한것과 다르게 return을 생략이 가능하고, 보통은 매개변수에 소괄호를 해야하는데, 매개변수가 하나일때는 괄호를 생략할 수 있습니다.

## 배열
배열은 기존의 다른언어들과 비슷하게 여러개의 데이터를 저장하지만, 배열은 자바스크립트에서 객체 취급을 받는다. 자바에서 객체는 key-value로 이루어져있는데 그러면 배열은 어떻게 되어있을까? 배열은 자바스크립트에서는 key가 인덱스번호가 되고 value는 데이터형식으로 표현된다. 배열관련 함수로는 map(): 배열의 값을 하나씩 바꿔서 새 배열을 만드는 함수, filter(): 조건에 맞는 요소만 추출하는 함수 some(): 배열에 하나라도 조건을 만족하는 게 있는지 확인할 때 사용하는 함수가 여러가지 있으며, 자바와 같은 역할을 하는 배열명.length 함수도 있다.
