## 변수에 대해 설명하고 let, const, var 차이점을 작성해주세요

1. 변수란?
   : 값(데이터)을 메모리에 저장하고 변수명으로 사용할 수 있는 것.

2. let, const, var 차이점
   -> 모두 변수를 선언하는 방식

| 구분         | var (옛날 방식) | let           | const            |
| ------------ | --------------- | ------------- | ---------------- |
| 재선언       | O               | X             | X                |
| 재할당       | O               | O             | X                |
| 범위 (Scope) | 함수 단위       | 블록 단위 { } | 블록 단위 { }    |
| 초기값       | 없어도 됨       | 없어도 됨     | 반드시 있어야 함 |

- var 사용 당시 상황 : 코드가 별로 길지도 않고 복잡하지 않음.
  크기가 커지면서 선언한 변수를 사용해 데이터를 잃고 오류가 나는 경우가 발생하여 재선언을 막는 방식으로 변함.

- 블록단위 : 조건문(if), 반복문(for) 등 프로그램 전체에 영향을 주지 않는 변수들을 사용할때조차 구분을 해줘야된다면 복잡하고 어려울 것. 그래서 중괄호 안에서만, 단위 안에서 동작하도록 한 것.

- const 반드시 초기값이 있어야 되는 이유 : 재할당이 안되기때문에 초기에 값을 넣어주지 않으면 끝까지 값이 없어서 쓸모가 없게 되기 때문이다.

3. 실전 사용법
   대부분 값을 바꾸지 않기때문에 const 사용, let은 변하는 let,for등에서 사용

## 주제1 : 콜백함수 = 함수의 인자로 전달되는 함수

콜백함수란?
앞서 함수 표현식에서 함수를 값으로 취급해 전달할 수 있는 것과 같이 함수는 다른 함수의 인수로도 전달할 수 있다는 것.
이를 콜백함수라고 한다.

```js
function parent(callback) {
  console.log("first");
  callback();
}

function child() {
  console.log("second");
}

parent(child); // 함수를 인수로 전달함

//출력값
//first
//second
```

콜박 함수 사용 이유 : 일을 병렬적으로 효율적이게 처리하고 싶은 경우 사용가능

```js
function repeat(count) {
  for (let idx = 0; idx < count; idx++) {
    console.log(idx + 1);
  }
}
repeat(5);

function repeat2(count) {
  for (let idx = 0; idx < count; idx++) {
    console.log((idx + 1) * 2);
  }
}

repeat2(5);
//  코드에 겹치는 부분이 많음
```

```js
function repeat(count, callBack) {
  // ③
  for (let idx = 0; idx < count; idx++) {
    callBack(idx + 1);
  }
}

function origin(count) {
  console.log(count);
}

function repeat2(count) {
  // ①
  console.log(count * 2);
}

repeat(5, repeat2);

// 출력 : 2 4 6 8 10

//함수 호출과 함께 사용가능 -이런경우 엄청 많음
repeat(5, function (count) {
  // ①
  console.log(count * 2);
});
```

## 주제2 : 배열과 배열관련 함수

배열을 변형하거나 요소를 재정렬하는 메서드 : 변형메서드

map() : 배열의 값을 순차적으로 돌면서 해당하는 값에 대해서 연산을 처리하여 새로운 배열을 만드는 함수

```js
let arr = [1, 2, 3];
let arr2 = arr.map((x) => x * 3);

console.log(arr2); //[3,6,9]
```

filter() : filter 거른다는 의미, 조건에 맞는 것만 배열에서 구하고 싶다면 조건을 주면 그에 해당하는 값만 추출하여 배열을 생성함

```js
let arr = [1, 2, 3, 4, 5];
let arr2 = arr.filter((x) => x > 3);
```

-> 둘 모두 연산의 결과는 배열임

some() : 배열에 하나라도 조건을 만족하는게 있는지 확인할 때 사용 -> 이것만 반환 값은 boolean형식

## 주제3 : 객체

객체 - key-value값을 한 쌍으로 묶은 덩어리들로 이루어짐
프로퍼티 : 키:값 한 쌍으로 묶은 덩어리들로 이루어져있음.

객체의 정보 사용하기

1. 점 표기법
2. 대괄호 표기법

- 배열과 함수도 객체인 이유
  : js의 원시자료형을 제외한 모든 자료형은 객체 자료형이다.
  : 배열 : 인덱스 - 값

```js
// Object.assign()
const arr = ["사과", "바나나", "딸기"];
const obj = Object.assign({}, arr);
console.log(obj); // {0: '사과', 1: '바나나', 2: '딸기'}
```
