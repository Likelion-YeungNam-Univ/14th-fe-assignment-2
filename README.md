## 변수에 대해 설명하고 let, const, var 차이점을 작성해주세요
변수는 컴퓨터가 명령을 처리하는 도중 발생하는 값을 저장하기 위한 공간이다.
정수형, 실수형, 문자형, 포인터형 등 모든 데이터 형식을 담을 수 있다.

var은 중복 선언과 재할당 모두 가능하고,
let은 중복 선언을 불가능하지만 변수에 재할당이 가능하고,
const는 변수 재선언, 변수 재할당 모두 불가능하다.

## ==과, ===의 차이점을 작성해주세요
== : 피연산자의 자료형이 달라도 값이 같으면 true를 반환한다.
=== : 피연산자의 값과 자료형이 같아야 true를 반환한다.

## var 변수 타입을 지양해야하는 이유를 말해주세요
코드량이 많아지면 값이 바뀔 수도 있고 어디서 어떻게 사용되는지 파악하기 힘들기 때문이다.

## function foo() {}와 var foo = function() {} 사이에서 foo 사용의 차이에 대해 작성해주세요
fucntion foo() {} 는 전체 함수가 호이스팅 되어 함수를 선언하기 전에 호출해도 적상적으로 작동한다.
var foo = function() {} 는 var만 호이스팅 되고 함수 할당은 해당 중에서 이루어진다. 선언 전에 호출하면 undefined 상태이기 때문에 오류가 발생한다.

## 화살표 => 함수 문법에 대한 사용 예시를 작성해주세요. 이 문법은 다른 함수와 어떻게 다른지 작성해주세요
const add = (a,b) => a + b;
console.log(add(10,20));

결과 : 30

화살표 함수를 사용하면 function 키워드를 사용해서 함수를 만드는 것보다 더 간결하게 표현할 수 있고 this, arguments, 생성자 함수가 없다.

## spread 문법에 대해 작성해주세요
spread는 ...기호를 사용하여 배열이나 객체를 펼쳐서 개별 요소로 분리해주는 연산자이다.
배열을 객체로 만들 때 {...arr}와 같이 사용하면 배열의 인덱스가 객체의 키가 되어 데이터가 저장된다.

## 주제1 (ex. 배열)
* 화살표 함수
: function 사용한 함수보다 더 간단한 함수 표현 가능
function sum(a, b){
    return a + b;
}

-> const add = (a, b) => a + b;

- 매개변수가 하나일 때 : () 생략 가능 / const add = a => a + a;
- 본문이 한 줄일 때 : {}, return 생략 가능 / const greet = name => console.log(name);

## 주제2 (ex. 객체)
* 객체 

1. object.keys() : 객체의 모든 속성(key)만 반환 - 반환 방식 : 배열
=> ["name", "age", "isAudult"]

2. object.values() : 객체의 모든 값(value)만 반환 - 반환 방식 : 배열 
=> ["철수", "18", "true"]

3. objects.entries() : 객체의 속성(key), 값(values) 함께 반환 - 반환 방식 : 배열
=>  0 : ["name", "철수"]
    1 : ["age", "18"]
    2 : ["isAdult", "true"]

4. Object.assign(target, sources) : 두 객체의 속성을 합칠 때 사용
target : 속성 받을 객체 / sources : 복사할 하나 이상의 객체 

< 배열 -> 객체 >
const fruits = [ "사과", "바나나" ];
const obj = Object.assign({}, fruits);
console.log(obj);

=> { 0:"사과", 1:"바나나" }

## 주제3 (ex.화살표 함수)
* 콜백함수 : 함수의 인자로 전달되는 함수
fucntion greet(name, callback){
    console.log("안녕하세요. " + name + "님!");
    callback(); -> 콜백함수 실행
