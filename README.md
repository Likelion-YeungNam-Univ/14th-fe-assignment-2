## 자바스크립트 언어가 다른 언어와 다른 특징 3가지를 작성해주세요

-> 자바스크립트 언어가 다른 언어와의 차이점은 첫번째로 자바스크립트는 동적 타이핑 방식을 사용하므로 변수 선언 시 let, var등을 사용하면 문자, 숫자 등을 알아서 인식을 하므로 int, char과 같은 형태의 변수선언을 따로 해줄 필요가 없습니다. 두번째로 함수를 값 처럼 취급할 수 있습니다. 값처럼 취급이 가능해서 함수를 매개변수 처럼 쓸 수도 있고, 다른 변수에 대입도 가능하고, 어떤 함수의 결과값으로도 사용이 가능합니다. 마지막으로 세번째는 비동기 처리입니다. 자바스크립트는 다른 언어와 달리 이벤트 루프라는 것을 이용하여 특정 작업이 진행되는 동안 멈춰있지 않고 다른 작업을 할 수 있어 효율적입니다.

## 변수에 대해 설명하고 let, const, var 차이점을 작성해주세요

-> 우선 let의 경우 중복선언이 불가능하며, 재할당이 가능한 변수로 제일 많이 쓰이는 변수 중 하나입니다. const는 중복 선언도 불가능하고 재할당도 불가능하여 한 번 생성되면 수정이 불가능합니다. 마지막으로 var의 경우 중복선언이 가능하고, 재할당 또한 가능합니다만 처음 선언할 때와 중복선언할 때의 차이가 없어 자신이 생성했던 이름의 변수 값이 중복선언으로 인해 예기치 못한 불상사가 일어날 수도 있습니다.

## var 변수 타입을 지양해야하는 이유를 말해주세요

-> var는 중복선언이 가능해서 이미 변수가 생성된 상태에서 다시 한 번 생성하는 불상사가 발생해도 알지 못하지만 let은  중복선언을 할 경우 Syntax Error가 발생해 개발자가 선언을 중복으로 불상사가 일어날 일이 없어집니다.

## function foo() {}와 var foo = function() {} 사이에서 foo 사용의 차이에 대해 작성해주세요


-> 전자의 경우 foo()라는 함수를 선언하는 코드이고, 후자의 경우는 var로 선언한 foo라는 변수에 function()이라는 함수를 집어넣은 것입니다. 전자의 경우 foo라는 함수 뒤에 {}블럭이 있어서 바로 실행이 되지만 후자의 경우 foo라는 변수에 function()이라는 함수를 집어넣는거라서 만약 function() 뒤에 블럭이 바로 오지 않고 foo가 실행이 되면 function()에 호이스팅이 되지 않아서 오류가 발생할 수도 있습니다.
